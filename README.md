# Passive Aggressive Translator

> Corporate speak decoded. Passive aggression exposed.

A Chrome extension that translates office jargon and passive-aggressive language into plain, honest English. Paste any email or Slack message — get back what they actually mean, the subtext, and exactly how to respond.

![Demo](demo.gif)

## What it does

- **What They Actually Mean** — blunt, honest translation with no corporate softening
- **Subtext** — power dynamics, avoidance, and manipulation explained
- **How to Respond** — a direct reply that doesn't play their game

## How to use

1. Click the extension icon
2. Paste any email, Slack message, or meeting note
3. Hit **▶ Translate**
4. Get the unfiltered truth

## Setup

1. Load the extension in Chrome (`chrome://extensions` → Developer Mode → Load unpacked)
2. Click ⚙ and paste your [Gemini API key](https://aistudio.google.com/apikey)
3. Done — works on any text you paste

## Tech

- Chrome Extension Manifest V3
- Google Gemini API (with OpenRouter fallback)
